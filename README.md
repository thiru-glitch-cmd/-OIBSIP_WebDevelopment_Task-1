# BiteBuddy - Food Delivery Landing Page

A modern, responsive, and visually appealing landing page for a food delivery web application called **BiteBuddy**, built using clean semantic HTML5 and vanilla CSS.

## Objective
The goal of this project was to design and implement a premium, interactive, and responsive landing page for "BiteBuddy", a food delivery service. The page aims to showcase popular dishes, highlight key service features (like fast delivery, fresh ingredients, secure payments), present customer reviews, and encourage users to download the official application or place an order.

---

## Key Features
- **Responsive Navigation Bar**: Includes links to different sections (Home, Menu, Features, Reviews, Contact) and a Call-to-Action (CTA) button that adapts seamlessly to mobile devices.
- **Hero Section**: Introduces the main value proposition with dynamic text, call-to-action buttons, and high-quality food imagery.
- **Popular Dishes Grid**: Displays curated popular dishes (Cheese Burger, Italian Pizza, Creamy Pasta, Fresh Sushi) with images, prices, and an interactive "Add to Cart" button.
- **Features Section**: Highlight cards outlining why users should choose BiteBuddy, enhanced with custom icons.
- **Testimonial Section**: Displays social proof via customer reviews.
- **App Download Banner**: Prompts users to download the app on Google Play or the App Store.
- **Footer**: Contains contact information, social links, and copyright text.

---

## Steps Performed
1. **HTML Architecture**: Constructed a semantic HTML structure using modern elements (`<header>`, `<nav>`, `<section>`, `<footer>`) to ensure accessibility and SEO compliance.
2. **CSS Styling**: Styled the elements using vanilla CSS, establishing a vibrant food-themed color palette, custom layout alignments, and a consistent typography hierarchy.
3. **Responsive Design Integration**: Implemented CSS media queries (specifically targeting screen widths below `768px`) to dynamically adjust layouts, convert horizontal listings into vertical stacks, and optimize spacing on mobile devices.
4. **Interactive Accents**: Programmed smooth hover states on call-to-action buttons and interactive cards to elevate user engagement.

---

## Tools & Technologies Used
- **HTML5**: Core structure and semantic markup.
- **CSS3**: Stylesheet layout management, media queries for responsiveness, and micro-interactions/hover animations.
- **Font Awesome (v6.5.1)**: Imported via CDN to display premium scalable vector icons.
- **Google Fonts & System Typography**: Selected clean sans-serif typefaces for maximum legibility.

---

## Outcome
A lightweight, modern, and production-ready landing page that delivers a premium visual experience. The interface scales cleanly from small mobile displays up to widescreen monitors, maintaining structural integrity, proper alignment, and fast load times.
